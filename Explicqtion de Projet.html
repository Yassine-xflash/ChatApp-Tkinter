Projet de Traitement de Signal

HeareFirst représente une application de chat de type “WhatsApp-like” utilisant Python et plusieurs bibliothèques. L’application permet l’inscription et la connexion via la voix, et dispose de fonctionnalités de chat avec synthèse vocale et enregistrement audio.

Bibliothèques utilisée

  • tkinter: Pour l’interface graphique.
  • messagebox: Pour les boîtes de dialogue.
  • PIL (Image, ImageTk): Pour la manipulation d’images.
  • sounddevice: Pour l’enregistrement audio.
  • numpy: Pour les opérations sur les tableaux.
  • scipy.io.wavfile: Pour lire et écrire des fichiers WAV.
  • threading: Pour gérer les threads.
  • tempfile: Pour créer des fichiers temporaires.
  • wave: Pour lire les fichiers audio.
  • pyaudio: Pour jouer des fichiers audio.
  • os: Pour les opérations sur le système de fichiers.
  • pyttsx3: Pour la synthèse vocale.
  • pygame: Pour l’audio.
  • soundfile: Pour lire et écrire des fichiers sonores.
  • librosa: Pour l’analyse audio.
  • sklearn.metrics.pairwise: Pour calculer la similarité cosinus.
  • shutil: Pour copier des fichiers.
  • speech_recognition: Pour la reconnaissance vocale.
  • scipy.signal (butter, lfilter): Pour le filtrage du signal audio.

Méthodes et Fonctionnalités

  1. init

    • Initialise la fenêtre principale et configure les propriétés de base.
    • Initialise le moteur TTS et les cadres de connexion et d’inscription.

  2. create_login_signup_frames

    • Crée et configure les cadres de connexion et d’inscription avec les widgets nécessaires (entrées, boutons, labels).

  3. show_signup et show_login

    • Affichent respectivement les cadres d’inscription et de connexion.

  4. record_signup_voice et record_login_voice

    • Enregistre la voix pour l’inscription et la connexion.

  5. record_voice

    • Enregistre l’audio pendant une durée définie et enregistre le fichier temporairement.

  6. signup

    • Enregistre les détails de l’utilisateur et copie l’enregistrement vocal dans un dossier spécifique.

  7. login

    • Valide les informations de connexion et compare les enregistrements vocaux pour l’authentification.

  8. extract_mfcc

    • Extrait les caractéristiques MFCC (Mel-frequency cepstral coefficients) d’un fichier audio.

  9. compare_voices

    • Compare les enregistrements vocaux en utilisant la similarité cosinus des caractéristiques MFCC.

  10. identify_speaker

    • Identifie le locuteur à partir d’un fichier audio en comparant les enregistrements vocaux existants.

  11. show_chat

    • Affiche l’interface de chat après une connexion réussie.

  12. create_chat_frame

    • Crée et configure le cadre de chat avec les widgets nécessaires pour l’affichage des messages et l’enregistrement audio.

  13. set_tts_properties

    • Configure les propriétés du moteur de synthèse vocale.

  14. send_message

    • Envoie un message texte et utilise la synthèse vocale pour lire le message.

  15. text_to_speech

    • Convertit du texte en parole.

  16. start_recording et record_audio

    • Gèrent l’enregistrement audio à partir de l’interface de chat.

  17. save_audio

    • Sauvegarde l’audio enregistré et affiche le message audio dans le chat.

  18. play_audio

    • Joue un fichier audio à partir d’un bouton de lecture.

  19. butter_bandpass et bandpass_filter

    • Définissent et appliquent un filtre passe-bande Butterworth pour le prétraitement audio.

  20. normalize_audio

    • Normalise les données audio.

  21. preprocess_audio

    • Prétravaille un fichier audio en appliquant un filtre passe-bande et en normalisant l’audio.

  22. speech_to_text

    • Convertit un fichier audio en texte en utilisant la reconnaissance vocale Google.

  23. display_audio_message

    • Affiche un message audio dans le chat avec un bouton de lecture.

  24. display_message

    • Affiche un message texte dans le chat.

Fonctionnalités Principaux

1. Connexion

Fonctionnalités :

  • Remplissage du formulaire de connexion : L’utilisateur entre son nom d’utilisateur et son mot de passe dans les champs de texte.
  • Enregistrement de la voix : L’utilisateur clique sur le bouton “Enregistrer la voix” pour enregistrer un échantillon vocal de 10 secondes, utilisé pour l’authentification vocale.
  • Vérification des détails : L’application vérifie si les détails saisis correspondent aux informations stockées.
  • Comparaison des voix : Les échantillons vocaux sont comparés à l’aide de caractéristiques MFCC (Mel Frequency Cepstral Coefficients) et de la similarité cosinus pour vérifier l’authenticité de l’utilisateur.
  • Connexion réussie ou échouée : En cas de correspondance, l’utilisateur est connecté et accède à l’interface de chat. Sinon, un message d’erreur est affiché.

2. Inscription

Fonctionnalités :

  • Remplissage du formulaire d’inscription : L’utilisateur entre son nom d’utilisateur et son mot de passe dans les champs de texte.
  • Enregistrement de la voix : L’utilisateur clique sur le bouton “Enregistrer la voix” pour enregistrer un échantillon vocal de 10 secondes.
  • Enregistrement des détails : Les informations de l’utilisateur (nom d’utilisateur, mot de passe et échantillon vocal) sont sauvegardées dans un répertoire local.
  • Confirmation de l’inscription : L’utilisateur est informé de la réussite de l’inscription et peut maintenant se connecter.

Diagramme Mermaid qui illustre le principe

3. Envoi de texte

Fonctionnalités :

  • Saisie de message : L’utilisateur saisit un message texte dans un champ de texte.
  • Envoi du message : L’utilisateur clique sur le bouton d’envoi pour envoyer le message.
  • Affichage du message : Le message est affiché dans la zone de chat avec une mention de l’expéditeur.

4. Transformation du texte en audio

Fonctionnalités :

  • Text-to-Speech (TTS) : L’application utilise la bibliothèque pyttsx3 pour convertir le texte en parole. Le moteur TTS lit le texte saisi par l’utilisateur à haute voix.
  • Propriétés TTS : Les propriétés telles que la voix (genre), la vitesse et le volume peuvent être configurées.

Diagramme Mermaid qui illustre le principe

5. Transformation de l’audio en texte

Fonctionnalités :

  • Enregistrement audio : L’utilisateur peut enregistrer un message vocal en cliquant sur le bouton d’enregistrement.
  • Reconnaissance vocale : L’application utilise la bibliothèque SpeechRecognition pour convertir l’audio en texte.
  • Prétraitement de l’audio : L’audio est prétraité à l’aide d’un filtre passe-bande et normalisé avant la reconnaissance.
  • Affichage du message : Le texte reconnu est affiché dans la zone de chat avec une mention de l’expéditeur.
